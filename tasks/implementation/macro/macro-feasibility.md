## 二、宏观指标含义分析

### 2.1 黄金 (GOLD)

- **战争/地缘风险**: 黄金是传统避险资产，地缘冲突升级时资金涌入黄金推高价格
- **通胀预期**: 黄金被视为通胀对冲工具，通胀预期上升利好黄金
- **实际利率**: 实际利率 = 名义利率 - 通胀预期。实际利率下降 → 黄金上涨；反之亦然
- **美元强弱**: 黄金以美元计价，美元走弱通常利好黄金
- **央行购金**: 全球央行增持黄金储备是近年结构性支撑因素

### 2.2 原油 (OIL)

- **通胀传导**: 原油是最核心的大宗商品，油价上涨直接推高 CPI
- **地缘风险溢价**: 中东局势、OPEC 减产协议直接影响供给预期
- **经济活动**: 原油需求反映全球制造业和运输业活跃度
- **战争因素**: 产油区冲突（中东、俄乌）导致供给中断风险
- **风险偏好**: 油价大跌往往伴随经济衰退预期（RISK OFF 信号）

### 2.3 比特币 (BTC)

- **风险偏好**: BTC 是高风险偏好的代表资产，RISK ON 时表现突出
- **流动性指标**: 全球流动性宽松时 BTC 往往大涨
- **数字黄金叙事**: 部分投资者将 BTC 视为"数字黄金"，与黄金有一定替代关系
- **监管因素**: 各国监管政策变化对 BTC 价格影响显著
- **机构化趋势**: BTC ETF 通过后机构资金流入是结构性变化

### 2.4 以太坊 (ETH)

- **Web3 生态**: ETH 是智能合约平台龙头，反映区块链生态活跃度
- **风险偏好**: 比 BTC 波动更大，是更高 Beta 的风险资产
- **DeFi 基础设施**: ETH 是 DeFi 生态的结算层
- **ETH/BTC 汇率**: ETH/BTC 比率变化反映市场对 altcoin 风险偏好的变化

### 2.5 利率 (Interest Rates)

- **联邦基金利率 (DFF)**: 美联储核心政策利率，直接影响全球资金成本
- **国债收益率曲线**:
  - 2年期 (DGS2): 短期利率预期
  - 10年期 (DGS10): 长期增长与通胀预期
  - 10Y-2Y利差 (T10Y2Y): 收益率曲线倒挂是衰退预警信号
- **实际利率**: 名义利率 - 通胀预期，决定黄金和风险资产定价
- **利率与风险资产**: 加息周期通常压制风险资产，降息周期利好

## 三、数据源可行性分析

### 3.1 HyperLiquid API（加密货币）

| 项目 | 详情 |
|------|------|
| 端点 | `POST https://api.hyperliquid.xyz/info` |
| 认证 | **无需认证**，读取接口完全公开 |
| 费用 | **完全免费** |
| 速率限制 | 1200 权重/分钟（`allMids` 仅 2 权重，约 600 请求/分钟） |
| 数据 | BTC、ETH 实时中间价、标记价、Oracle 价格、资金费率、OI、24h 成交量 |
| 格式 | JSON，价格为字符串 |

**获取所有中间价示例**:
```bash
curl -X POST https://api.hyperliquid.xyz/info \
  -H "Content-Type: application/json" \
  -d '{"type": "allMids"}'
# 返回: {"BTC": "82345.5", "ETH": "1920.25", ...}
```

**获取永续合约详情**:
```bash
curl -X POST https://api.hyperliquid.xyz/info \
  -d '{"type": "metaAndAssetCtxs"}'
# 返回: markPx, oraclePx, prevDayPx, funding, openInterest, dayNtlVlm
```

**结论: 完全可行，无需任何注册或费用。**

### 3.2 FRED API（经济数据）

| 项目 | 详情 |
|------|------|
| 端点 | `https://api.stlouisfed.org/fred/series/observations` |
| 认证 | 需要 API Key（免费注册，即时获取） |
| 费用 | **完全免费** |
| 速率限制 | 宽松，足以支持本项目 |
| 格式 | JSON / XML / CSV |

**关键数据系列**:

| Series ID | 描述 | 频率 |
|-----------|------|------|
| `DFF` | 联邦基金有效利率 | 日 |
| `DGS2` | 2年期国债收益率 | 日 |
| `DGS10` | 10年期国债收益率 | 日 |
| `DGS30` | 30年期国债收益率 | 日 |
| `T10Y2Y` | 10Y-2Y 收益率利差 | 日 |
| `GOLDAMGBD228NLBM` | 伦敦金上午定盘价 (USD/oz) | 日 |
| `DCOILWTICO` | WTI 原油现货价 (USD/barrel) | 日 |

**结论: 完全可行，一个 API Key 覆盖利率、黄金、原油三大类数据。**

### 3.3 数据延迟对比

| 数据源 | 实时性 | 适用场景 |
|--------|--------|----------|
| HyperLiquid | 实时（秒级） | BTC/ETH 即时价格 |
| FRED - 利率 | T+1（每日更新） | 利率趋势分析 |
| FRED - 黄金 | T+1（每日更新） | 黄金趋势分析 |
| FRED - 原油 | T+1（每日更新） | 原油趋势分析 |

> **补充方案**: 如需实时黄金/原油价格，可后续集成 Commodities-API 或 CoinGecko 作为补充。

## 四、RISK ON / RISK OFF 判定模型

### 4.1 判定因子

| 因子 | 数据 | RISK ON 信号 | RISK OFF 信号 |
|------|------|-------------|--------------|
| 原油趋势 | OIL 7日变化 | 油价稳定/温和上涨 | 油价暴跌（需求坍塌） |
| 黄金趋势 | GOLD 7日变化 | 金价走平/下跌 | 金价急涨（避险涌入） |
| BTC 强度 | BTC 7日变化 | BTC 上涨 | BTC 下跌 |
| 收益率曲线 | T10Y2Y | 曲线正常/陡峭 | 曲线倒挂 |
| 波动率 | 各资产 20日波动率 | 低波动 | 高波动 |

### 4.2 综合评分模型

```
Risk Score = w1 * Oil_Signal + w2 * Gold_Signal + w3 * BTC_Signal
           + w4 * YieldCurve_Signal + w5 * Volatility_Signal

权重 (初始):
  w1 (原油) = 0.20
  w2 (黄金) = 0.25
  w3 (BTC)  = 0.25
  w4 (利差) = 0.20
  w5 (波动) = 0.10
```

### 4.3 模式判定阈值

| Risk Score | 模式 | 描述 |
|-----------|------|------|
| ≥ 0.6 | RISK ON | 市场风险偏好高，利好风险资产 |
| 0.4 - 0.6 | NEUTRAL | 观望，保持现有仓位 |
| < 0.4 | RISK OFF | 市场避险情绪浓厚，利好避险资产 |

### 4.4 操作建议

**RISK ON 模式**:
- 增持 BTC、ETH 等风险资产
- 减少黄金、现金配置
- 关注动量策略
- 可适度使用杠杆

**RISK OFF 模式**:
- 增持黄金
- 增加现金比例
- 减持 BTC、ETH
- 避免高杠杆
- 关注国债等安全资产

**NEUTRAL 模式**:
- 保持均衡配置
- 等待明确信号
- 定投策略适用
